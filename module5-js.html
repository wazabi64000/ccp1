Module 5 : JavaScript — JSON externe, DOM, forEach et LocalStorage

1. Concepts importants

  • DOM : interface représentant la page. Chaque élément HTML devient un nœud manipulable par JavaScript.
  • JSON : format de données pour représenter des objets ou tableaux (ex. annonces).
  • forEach : méthode pour parcourir un tableau et exécuter une fonction sur chaque élément.
  • LocalStorage : stockage persistant dans le navigateur pour conserver les favoris ou autres données.
  • ES6 : let/const, arrow functions, template literals, import/export pour modules.
Astuce : Utiliser ES6 simplifie le code et le rend plus lisible.

3. LocalStorage


// Récupérer les favoris
let favoris = JSON.parse(localStorage.getItem('favoris')) || [];

// Ajouter ou retirer un favori
const toggleFavori = (id) => {
  if(favoris.includes(id)){
    favoris = favoris.filter(f => f !== id);
  } else {
    favoris.push(id);
  }
  localStorage.setItem('favoris', JSON.stringify(favoris));
};
Explication : LocalStorage conserve les favoris même après la fermeture du navigateur.
