Projet final : site d'annonces

Fonctionnalités : affichage dynamique, recherche, tri, filtres, favoris avec LocalStorage.

Favoris

Vos favoris sont conservés localement dans le navigateur.

Explication détaillée étape par étape

  1. Données JSON : Nous créons un tableau d’objets `annonces` contenant toutes les informations pour chaque annonce : id, titre, prix, categorie, image, date. Cela simule un fichier JSON externe ou une API.
  2. LocalStorage : La fonction getFavoris() récupère les favoris stockés dans le navigateur, et toggleFavori(id) ajoute ou retire un favori et met à jour le localStorage. Cela permet de conserver les favoris même après fermeture du navigateur.
  3. Render Annonces : La fonction renderAnnonces() :
    • Récupère la recherche texte, la catégorie sélectionnée et le type de tri.
    • Filtre le tableau JSON en fonction de ces critères.
    • Trie les annonces si nécessaire (prix croissant/décroissant, date récente).
    • Génère le HTML de chaque carte dynamiquement et l’injecte dans #annoncesContainer.
    • Affiche le bouton “Ajouter/Retirer des favoris” selon si l’annonce est dans le localStorage.
  4. Render Favoris : La fonction renderFavorisUI() :
    • Récupère les IDs des favoris dans le localStorage.
    • Filtre les annonces correspondant à ces IDs.
    • Affiche chaque favori dans #favorisList avec image, titre, catégorie, date et bouton “Supprimer”.
  5. Initialisation : La fonction initUI() :
    • Génère automatiquement les options de catégories dans le select.
    • Appelle renderAnnonces() et renderFavorisUI() pour afficher les données au chargement.
    • Ajoute des listeners sur les inputs de recherche, sélection de catégorie et tri pour mettre à jour l’affichage en direct.
  6. Événements : Les boutons “Ajouter/Retirer des favoris” appellent toggleFavori(id) qui met à jour à la fois la liste des annonces et la liste des favoris.
  7. Accessibilité : Chaque image possède un attribut alt, le conteneur des annonces utilise aria-live="polite" pour signaler les changements aux lecteurs d’écran, et le site est navigable au clavier.
