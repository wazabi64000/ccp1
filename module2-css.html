Module 2 : CSS — Sélecteurs, Flexbox et Grid

1. Fondamentaux CSS

CSS (Cascading Style Sheets) permet de définir le style visuel d’une page HTML. Contrairement à HTML, qui décrit la structure et le sens du contenu, CSS contrôle :

  • La couleur et la typographie (font, color, line-height)
  • La taille et l’espacement (margin, padding, width, height)
  • L’affichage et le positionnement (display, position, flex, grid)
  • Les effets visuels (background, border, box-shadow)

On applique le CSS via :

  • Des sélecteurs par balise : h1, p, div
  • Des classes : .ma-classe
  • Des ID : #mon-id
  • Des sélecteurs combinés ou pseudo-classes : nav ul li:first-child
À retenir : Utilisez les classes pour la réutilisabilité et évitez d’utiliser trop d’ID pour le style.

2. Flexbox — Alignement et distribution

Flexbox est utilisé pour organiser les éléments sur un axe unique (ligne ou colonne) et gérer leur alignement et leur distribution de l’espace. Il est particulièrement utile pour :

  • Aligner les éléments horizontalement ou verticalement
  • Distribuer l’espace de manière équitable
  • Gérer des composants responsifs simples

Exemple :

Col 1
Col 2
Col 3

.flex-demo {
  display: flex;       /* active le mode flex sur le conteneur */
  gap: 0.6rem;         /* espace entre les éléments */
  justify-content: space-between; /* répartit les éléments horizontalement */
}
.flex-demo > div {
  flex: 1;             /* chaque enfant prend une part égale */
  padding: 1rem;
  text-align: center;
}

Explication : Le conteneur devient un flex container. Les enfants peuvent être alignés et dimensionnés facilement avec flex et justify-content.

3. CSS Grid — Mise en page 2D

CSS Grid est idéal pour gérer la mise en page sur deux dimensions : lignes et colonnes. Il permet de :

  • Créer des grilles avec un nombre fixe ou dynamique de colonnes
  • Gérer les espacements avec gap
  • Combiner lignes et colonnes pour des mises en page complexes

B
C
D

.grid-demo {
  display: grid;
  grid-template-columns: repeat(4, 1fr); /* 4 colonnes égales */
  gap: 0.6rem;
}
À retenir : Flexbox = 1D (axe unique), Grid = 2D (lignes + colonnes). Combinez-les pour des layouts avancés.

4. Bonnes pratiques CSS

  • Utiliser des classes modulables et réutilisables pour maintenir le CSS lisible et maintenable.
  • Grid pour la structure principale de la page, Flexbox pour les composants internes.
  • Utiliser des unités relatives (rem, %, vh, vw) pour les mises en page responsives.
  • Organiser le CSS par sections ou composants.
  • Commenter le code pour expliquer les choix de mise en page.

5. Exercices progressifs

Exercice 1 — Style de base
Créez un fichier style.css et appliquez :
  • Couleur de fond de la page
  • Couleur et police du texte
  • Espacement autour des paragraphes
Exercice 2 — Flexbox
Créez un conteneur avec plusieurs div enfants. - Transformez-le en flex container horizontal - Ajoutez un espace de 10px entre chaque élément - Centrez le contenu verticalement
Exercice 3 — Grid responsive
Créez une grille d’articles avec : display: grid; grid-template-columns: repeat(auto-fit, minmax(200px, 1fr)); Chaque élément doit être responsive et la grille doit s’adapter à la largeur du navigateur.
Indice pour l’exercice Grid

Utilisez auto-fit pour remplir automatiquement le conteneur et minmax pour définir une largeur minimale et maximale par colonne.

6. Quiz rapide

Quel layout est le plus adapté pour aligner des éléments sur une seule ligne ?

Défi bonus : Combinez Flexbox et Grid pour créer une page avec une grille d’articles (2D) et des composants internes alignés avec Flexbox.
