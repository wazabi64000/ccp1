Module 4 : Responsive Design avec Flexbox & Grid

1. Principes du Responsive Design

Le Responsive Design consiste à créer des pages web qui s’adaptent automatiquement à différentes tailles d’écran : desktop, tablette, mobile.

  • Définir la <meta name="viewport"> pour contrôler l’échelle et la largeur
  • Utiliser des unités relatives : %, em, rem, vw, vh
  • Appliquer Flexbox ou Grid pour gérer les mises en page dynamiques
  • Utiliser @media pour modifier le style selon la taille d’écran
À retenir : Le responsive améliore l’expérience utilisateur et l’accessibilité sur tous les appareils.

2. Flexbox — Organisation 1D

Flexbox est idéal pour organiser des éléments sur un axe unique (horizontal ou vertical).

Propriétés du conteneur

  • display: flex; — transforme un conteneur en flex container
  • flex-direction — axe principal : row ou column
  • justify-content — alignement sur l’axe principal : flex-start, center, space-between
  • align-items — alignement sur l’axe secondaire : stretch, center, flex-start
  • gap — espace entre les éléments

Exemple Flexbox

Box 2
Box 3

.demo {
  display: flex;
  flex-direction: row;       
  justify-content: space-between; 
  align-items: center;       
  gap: 10px;                 
}
Remarque : gap simplifie l'espacement, évitant l'utilisation manuelle de margin sur chaque élément.

Propriétés des enfants

  • flex: 1; — occupe une part égale de l’espace disponible
  • flex-grow, flex-shrink, flex-basis — contrôle fin de la flexibilité

3. CSS Grid — Organisation 2D

Grid permet de gérer un layout en 2 dimensions : lignes et colonnes.

Propriétés du conteneur

  • display: grid; — active le mode Grid
  • grid-template-columns, grid-template-rows — définissent la structure des colonnes et lignes
  • gap — espace entre lignes et colonnes

Unités importantes

  • fr — fraction de l’espace disponible (ex. 1fr 2fr)
  • auto — dimension automatique selon le contenu
  • minmax(min,max) — limite minimale et maximale d’une colonne
  • repeat(n,value) — répète une taille n fois

Exemple Grid classique

Colonne 2 (double)
Colonne 3

.demo {
  display: grid;
  grid-template-columns: 1fr 2fr 1fr;
  gap: 10px;
}

.demo {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(150px, 1fr));
  gap: 10px;
}
Explication : auto-fit crée autant de colonnes que possible et minmax(150px,1fr) assure une taille minimum de 150px et maximum proportionnelle.

5. Media Queries


/* Desktop → 4 colonnes */
.grid-demo{ grid-template-columns: repeat(4,1fr); }

/* Tablette */
@media (max-width:900px){ 
  .grid-demo{ grid-template-columns: repeat(2,1fr); } 
}

/* Mobile */
@media (max-width:600px){ 
  .grid-demo{ grid-template-columns: 1fr; } 
}
Commencez par le style desktop (desktop-first) ou mobile (mobile-first), puis adaptez avec @media.

6. Bonnes pratiques

  • Tester sur plusieurs tailles d’écran
  • Combiner unités relatives, Flexbox et Grid pour la mise en page
  • Ne pas cacher le contenu essentiel sur mobile
  • Prioriser contenu et accessibilité

7. Exercices pratiques

Exercice 1 — Grille responsive : Transformez une grille 3 colonnes pour qu’elle devienne 2 colonnes sur tablette et 1 colonne sur mobile.
Exercice 2 — Flexbox responsive : Créez un menu horizontal sur desktop, puis vertical sur mobile grâce à @media et flex-direction.
Exercice 3 — Image adaptative : Appliquez max-width:100% et height:auto pour qu’elles ne débordent pas sur mobile.
