Module 1 : HTML & Structure

1. Le rôle du HTML

HTML (HyperText Markup Language) définit la structure et la sémantique du contenu d'une page web. Les navigateurs lisent ces balises pour afficher le contenu, et les outils d’accessibilité s’en servent pour aider les utilisateurs.

À retenir : HTML n’est pas du design, mais la structure et le sens du contenu.

2. Doctype et structure minimale

Un document HTML commence par :

<!DOCTYPE html>
<html lang="fr">
  <head>
    <meta charset="UTF-8">
    <title>Titre de la page</title>
  </head>
  <body>
    <!-- Contenu ici -->
  </body>
</html>

Explication : Le <!DOCTYPE html> indique au navigateur de rendre la page en mode standard HTML5. L'attribut lang="fr" est important pour le SEO et l’accessibilité.

3. Structure sémantique basique

<header>
  <h1>Titre du site</h1>
</header>

<nav><!-- menu --></nav>

<main>
  <article>
    <h2>Titre d'article</h2>
    <p>Contenu...</p>
  </article>
  <aside>Informations complémentaires</aside>
</main>

<footer>Pied de page</footer>
Utilisez header, nav, main, article, aside, footer plutôt que des div génériques.

4. Bonnes pratiques

  • Balises sémantiques pour la lisibilité et l’accessibilité.
  • Attributs alt pour les images.
  • Ordre logique du DOM pour la navigation au clavier.
  • Meta viewport et charset pour l’affichage correct sur mobile.

5. Exercices

Exercice 1 — Structure minimale
Crée un fichier test.html avec la structure HTML minimale, puis ajoute un article et un aside.
Exercice 2 — Page de présentation
Crée une page avec : - un titre principal (<h1>), - deux sections (<section>) avec titres secondaires (<h2>) et paragraphes, - un menu de navigation (<nav>) avec au moins deux liens.
Exercice 3 — Mini-site 2 pages
- Page 1 : Accueil avec titre et lien vers Page 2. - Page 2 : À propos avec texte et lien retour. Astuce : utilisez <a href="page2.html"> pour relier les pages.
Indice pour les exercices Pour les liens utilisez <a href="...">, pour les images <img src="..." alt="...">.
