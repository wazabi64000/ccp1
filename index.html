Exemple de structure HTML de base

Un document HTML commence toujours par la déclaration <!DOCTYPE html>, suivie des balises <html>, <head> et <body>. 

<!DOCTYPE html>
<html lang="fr">
  <head>
    <meta charset="utf-8">
    <title>Mon premier document</title>
  </head>
  <body>
    <h1>Bonjour le web</h1>
    <p>Ceci est un paragraphe.</p>
  </body>
</html>
À retenir : - <h1> est le titre principal de la page. - <p> définit un paragraphe. - Toujours préciser lang="fr" pour l’accessibilité et le référencement.
Exercice 1 — Page CV
Créez une page contenant : - votre nom en <h1>, - une section <h2>Expérience</h2> avec au moins deux paragraphes, - une section <h2>Contact</h2> avec un email cliquable (<a href="mailto:...">).
Exercice 2 — Page Recette
Créez une page qui affiche une recette de cuisine : - un titre (<h1>), - une liste d’ingrédients (<ul>), - une liste numérotée des étapes (<ol>), - une image du plat (<img>).
Exercice 3 — Mini-site
Créez deux pages reliées : - Page 1 : Accueil avec un titre et un lien vers la page 2. - Page 2 : À propos avec un texte de présentation et un lien retour vers la page d’accueil. Astuce : utilisez <a href="page2.html"> pour relier les fichiers.
Indice pour les exercices Utilisez la balise <a href="..."> pour insérer un lien, et <img src="..." alt="..."> pour afficher une image.

Quiz : Quelle balise est utilisée pour créer une liste numérotée ?

Défi bonus : Créez un mini-site de trois pages reliées par un menu de navigation (Accueil, Recettes, Contact). - Ajoutez un en-tête commun avec <nav>. - Placez un pied de page (<footer>). - Utilisez <main> pour le contenu principal.
